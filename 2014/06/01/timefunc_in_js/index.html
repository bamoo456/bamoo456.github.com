timeFunc_in_Js

javasript中，system time的相關作法主要是透過built-in的date ojbect

主要的初始化func用法如下:

var date = new Date();
var date = new Date(msec);
var date = new Date(dateString);
var date = new Date(year, month, day, hours, minutes, seconds, milliseconds);

在new出date ojbect以後就可以透過其相對應的func來取得系統的許多跟時間相關的func

取得目前數字表示的timestamp

new Date().getTime();

Returns the number of milliseconds since midnight Jan 1, 1970

簡單講它會回傳一連串的數字來代表目前的時間，而這個對我們通常只是要記錄目前的時間點時，會是很好用的。

parse目前的date string

var d = new Date().toDateString();
d.parse();

Returns the number of milliseconds since midnight Jan 1, 1970

一樣是回傳一連串的字數代表timestamp，但不同的是，我們可以使用符合 date format 的字串來當作來源。

變更date object目前的時間

socket.on('timeEvt', function(date){
    var d = new Date();
    d.setTime(date.time);
});

假設目前我們取得server端送來的一段time number，則我們可以透過這個func 來產生對應的date object

產生符合本地時間的time string

var d = new Date();
var localTime = d.toLocaleTimeString();

由於我們產生出來的string都是以utc為主的中央標準時間；然而考量到user如果在不同的國家時，我們就應該產生對應的
local time來表示。
