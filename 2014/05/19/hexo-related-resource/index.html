hexo 相關的資源

hexo 真是個很特別的framework東西，用起來不難!
對於現在在玩nodejs的我來說，剛好是個把玩的好東西…

把玩的同時又可以寫寫blog，真是不錯 :)

簡易寫blog的方式

hexo new "the name of new article"

當要建立一份新的文章時，可以透過這個指令去讓hexo自動建立一份post
而文章的產生會在這個path下: “source/_posts/‘the name of new article’“

hexo server

我們可以透過這個指令，在本地端建立一個blog server，讓修改的文章被即時在local 端的browser所看到!
http://localhost:4000 就是本地端可以看到的網址!

hexo generate

在目前的文章或設定，只有任何變動的部分都要使用這個指令去讓hexo建立對應的gihub html

hexo deploy

最後當本地端的server 都確定沒問題了；就可以透過這個指令deploy到github上!
